## Company Mission

AetherForge Interactive exists to build competitive experiences that reward skill,
celebrate fair play, and bring players together across every region on Earth.

Eclipse Dominion is the embodiment of that mission: a live-service competitive
shooter where every match is decided by mastery, not wallet size.

---

## Core Principles

### 1. Player-First Competitive Experience

Every design, infrastructure, and monetization decision is evaluated against a
single question: *does this make the competitive experience better for players?*
If the answer is ambiguous, we default to the player.

### 2. Global Operation, Local Excellence

Five studios -- Los Angeles (HQ), Montreal, Tokyo, Berlin, and Warsaw -- operate
as a single product team distributed across four regions (NA, EU, APAC, LATAM).
Each studio owns a domain of expertise:

| Studio    | Domain                                        |
|-----------|-----------------------------------------------|
| LA        | Platform ops, infrastructure, esports, comms  |
| Montreal  | Game design, balance, narrative               |
| Tokyo     | APAC infrastructure, netcode, matchmaking     |
| Berlin    | Monetization, EU compliance, localization     |
| Warsaw    | QA, anti-cheat, regression testing            |

No studio operates in isolation. Cross-studio review is required for any change
that touches competitive integrity, monetization, or player-facing narrative.

### 3. Competitive Integrity Above All

Eclipse Dominion's ranked and tournament modes are governed by strict fairness
guarantees. Balance changes undergo Montreal-led review. Anti-cheat (VanguardShield)
is maintained by Warsaw with a contractual false-positive rate below 0.1%.
Tournament balance freezes are mandatory 72 hours before any sanctioned event.

### 4. Fair Monetization

Monetization supports the game without compromising competitive fairness.
Cosmetic-only items are the default. Any mechanic that could be perceived as
pay-to-win requires executive sign-off and a public rationale. Battle Pass
pricing is benchmarked against regional purchasing power parity.

### 5. Narrative-Driven Engagement

Eclipse Dominion's seasonal arcs are canon. Lore consistency across all regions
and languages is a hard requirement. Narrative changes go through Montreal's
editorial review gate before deployment to any live environment.
