# DLR-20260304-002: Competitive Integrity Reporting System

| **Domain**    | Competitive Fairness                       |

## Context

Eclipse Dominion's ranked mode (1.8M monthly active players) lacks automated anomaly detection for competitive integrity. Current enforcement relies on manual review of player reports (~14K/week), catching only 62% of confirmed bad actors. Win-rate outliers, suspicious input patterns (inhuman reaction times, pixel-perfect tracking), and rating manipulation go undetected for an average of 18 days. The anti-cheat upgrade (DLR-20260228-002) addresses client-side cheating; this system addresses behavioral and statistical anomalies that evade signature-based detection.

## Options Considered

1. **Automated reporting dashboard with statistical anomaly detection (selected)** -- real-time pipeline analyzing win-rate, K/D, input patterns, and rating trajectories. Flags outliers for review; high-confidence cases trigger automated temporary restrictions.
2. **Expand manual review team** -- hire 8 additional reviewers. Linear scaling, high ongoing cost ($960K/yr), does not improve detection quality.
3. **Community-driven tribunal system** -- peer review of flagged players. Low cost but slow (7-14 day cycle) and subject to bias.

## Decision

Adopt Option 1. Build an automated competitive integrity reporting system. Pipeline ingests ranked telemetry (K/D, win-rate, headshot %, input timings, rating trajectory) and flags statistical outliers (>3 sigma from skill-bracket norms). High-confidence flags (>95% probability) trigger automated 24h temporary restrictions pending human review. Dashboard provides real-time visibility to the integrity team.

## Trade-offs

- (+) Detection coverage increases from 62% to projected 89% of bad actors.
- (+) Mean time to restriction drops from 18 days to <24 hours for high-confidence cases.
- (-) False positives on legitimate high-skill players require rapid human review SLA.
- (-) $420K build cost with ongoing $85K/yr operational cost.

## Assumptions

- **ASM-0009**: Player base accepts automated temporary restrictions for flagged accounts (based on competitor precedent and community survey -- 78% approval for "ban first, appeal fast" model).

## Kill Switch / Rollback

Disable automated restrictions immediately; revert to manual-review-only mode. Dashboard and detection pipeline continue operating in observation-only mode (no enforcement). Kill switch is a single feature flag: `INTEGRITY_AUTO_ENFORCE=false`. Rollback time: <5 minutes.
