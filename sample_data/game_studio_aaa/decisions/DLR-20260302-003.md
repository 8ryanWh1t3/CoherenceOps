# DLR-20260302-003: Tournament-Specific Balance Patch Freeze

| Field         | Value                                      |
|---------------|--------------------------------------------|
| **Status**    | Sealed                                     |
| **Date**      | 2026-03-02                                 |
| **Owner**     | Gameplay Systems (Montreal)                |
| **Studios**   | Montreal (lead), LA (approval)             |
| **Domain**    | Gameplay Balance                           |

## Context

EDCS W10 finals are on 2026-03-14. Professional teams have been practicing on the current balance patch (v4.2.1) for 3 weeks. Any balance change within the tournament window would invalidate team strategies and undermine competitive integrity. The Weapon Balance Framework v2 (DLR-20260224-004) automated detection system may flag weapons during this period, but changes must be held.

## Options Considered

1. **72-hour hard freeze before and during event (selected)** -- no balance changes from 2026-03-11 00:00 UTC through 2026-03-16 00:00 UTC. Exception: game-breaking exploits only.
2. **48-hour freeze** -- tighter window. Allows a last-minute tune on 2026-03-12 but risks insufficient pro practice time.
3. **No freeze, pro tournament uses locked build** -- tournament runs on a snapshot build. Public continues receiving patches. Creates version divergence confusion.

## Decision

Adopt Option 1. Freeze all weapon, ability, and equipment balance changes from 2026-03-11 00:00 UTC through 2026-03-16 00:00 UTC (5-day window). The automated balance detection system (DLR-20260224-004) continues monitoring but suppresses auto-flagging during the freeze. Exception: sev-1 game-breaking exploits may be hotfixed via emergency process with tournament director + lead designer sign-off.

## Trade-offs

- (+) Guarantees competitive integrity for professional and ranked players.
- (+) Builds pro community trust in AetherForge's esports commitment.
- (-) Any balance outlier discovered during freeze persists for 5 days in public play.
- (-) Suppressing automated detection creates a brief blind spot.

## Assumptions

- **ASM-0011**: The current balance state (v4.2.1) has no known sev-1 exploits as of freeze start.

## Kill Switch / Rollback

Unfreeze immediately if a sev-1 exploit is confirmed. Unfreeze requires joint sign-off from: (1) EDCS Tournament Director, (2) Montreal Lead Designer, (3) LA VP of Production. Emergency hotfix follows Patch Tuesday SLA (DLR-20260228-001).
